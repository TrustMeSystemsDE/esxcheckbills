ESX Police Billing System
Das ESX Police Billing System ermöglicht es Polizisten, offene Rechnungen von Spielern in der Nähe zu überprüfen und diese direkt zu begleichen.

Features
Überprüfen Sie als Polizist die offenen Rechnungen jedes nahegelegenen Spielers.
Zwingen Sie Spieler dazu, ihre offenen Rechnungen direkt aus ihrem Bargeld zu begleichen.
Spieler erhalten automatische Benachrichtigungen, wenn eine Rechnung von einem Polizisten beglichen wurde.
Voraussetzungen
Ein laufender ESX Server
MySQL
Installationsanleitung
Download & Dateien Extrahieren

Laden Sie das Skript herunter und extrahieren Sie die Dateien in Ihr resources Verzeichnis.
Ressource in den Server einbinden

Fügen Sie start checkbills zu Ihrer server.cfg Datei hinzu.
Datenbank aktualisieren

Stellen Sie sicher, dass Ihre billing Tabelle in Ihrer ESX-Datenbank existiert. Dieses Skript nutzt diese Tabelle, um offene Rechnungen abzurufen.
Server neustarten

Starten Sie Ihren FiveM Server neu. Das Skript sollte nun funktionieren.
